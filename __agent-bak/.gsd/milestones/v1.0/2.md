# Phase 2: Home Page Experience

## Objective
Build the core landing page sections (Hero, Features, Pricing Widget) to showcase the SaaS starter's capabilities with a premium aesthetic.

## Tasks
- [ ] **Task 1: Hero Asset Generation**
  - Use `generate_image` to create a modern, sleek dashboard UI mockup suitable for a SaaS header.
- [ ] **Task 2: Implementation of Hero Section**
  - Use `Home.jsx` to build the Hero.
  - Features: Gradient tagline, CTA buttons (Get Started, View Demo), and the generated image.
  - Styling: Large typography, spacing, and a subtle background glow.
- [ ] **Task 3: Feature Grid Component**
  - Section title: "Everything you need to ship."
  - 4 Cards:
    - **Authentication**: Passport.js with Social OAuth.
    - **Payments**: Stripe Subscriptions & Trials.
    - **Architecture**: Express 5, React 19, MongoDB.
    - **DX**: Vite, ESLint, Dynamic Routing.
- [ ] **Task 4: 4-Tier Pricing Widget**
  - Section ID: `pricing`.
  - Tiers: Hobby ($0), Startup ($19), Pro ($49), Enterprise (Custom).
  - Features: Glassmorphism cards, checkmark icons, and a "Most Popular" badge for Pro.
- [ ] **Task 5: Anchor Navigation Logic**
  - Ensure the "Pricing" link in the header correctly scrolls to the `#pricing` section on the Home page.

## Verification Criteria
- [ ] Hero section is visually impactful with a clear CTA.
- [ ] Feature grid is responsive and clearly communicates value.
- [ ] Pricing section displays 4 distinct tiers with consistent styling.
- [ ] Clicking "Pricing" in the header scrolls to the pricing section.
- [ ] Visual polish (hover states, gradients) consistent with Phase 1.
