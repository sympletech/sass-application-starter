# Phase 1: Foundation & Header

## Objective
Establish the global design system and update the layout shell with a modern, premium navigation.

## Tasks
- [ ] **Task 1: Premium Design System (Global CSS)**
  - Create `@client/src/index.css` with CSS variables for:
    - `--premium-gradient`: Linear gradient for headers/buttons.
    - `--glass-bg`: Semi-transparent background for glassmorphism.
    - `--glass-blur`: 10px-15px backdrop-filter.
    - `--text-premium`: Polished font family and scale.
  - Import `index.css` in `@client/src/Main.jsx`.
- [ ] **Task 2: Route Expansion & Placeholders**
  - Create placeholder components:
    - `@client/src/pages/About.jsx`
    - `@client/src/pages/Legal.jsx`
  - Update `@client/src/Main.jsx` to register `/about` and `/legal`.
- [ ] **Task 3: Refactor PublicLayout Shell**
  - Update `Header` styling in `@client/src/layouts/public/PublicLayout.jsx`:
    - Remove inline solid background, replace with glassmorphism class.
    - Add sticky positioning and Z-index.
  - Update `menuItems` to include:
    - `{ key: '/about', label: <Link to="/about">About</Link> }`
    - `{ key: '#pricing', label: <a href="/#pricing">Pricing</a> }`
    - `{ key: '/legal', label: <Link to="/legal">Legal</Link> }`
  - Update Mobile Drawer with new links.
- [ ] **Task 4: Ant Design Theme Refinement**
  - Update `ConfigProvider` tokens in `PublicLayout.jsx` to match premium aesthetic (e.g., primary color, border radius).

## Verification Criteria
- [ ] Header has glassmorphism blur effect and stays sticky on scroll.
- [ ] All new menu items (About, Legal, Pricing) are visible in desktop and mobile views.
- [ ] Navigating to `/about` and `/legal` shows the placeholder content.
- [ ] Browser console shows no routing or style loading errors.
